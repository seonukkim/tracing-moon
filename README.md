# Tracing the Moon: A Survey on TYPE-MOON

This repository contains the final PDF editions of *Tracing the Moon: A Survey on TYPE-MOON* by Seonuk Kim.

The paper surveys the publication history, media-mix development, and continuity structure of TYPE-MOON works, including *Tsukihime*, *Fate*, *Mahoutsukai no Yoru*, and *Kara no Kyoukai*. Rather than treating the catalog as a single timeline, it organizes relationships among original works, remakes, adaptations, parallel branches, and derivative works.
